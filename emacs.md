Emacs
===

Help on a key:

`C-h k`
or
`C-h c <key sequence>`

Find key shortcut for a function

`C-h function-name`
or
`C-h w function-name`

Filter lines:

```
M-x keep-lines
M-x filter-lines
```

Join lines with a delimiter e.g. ",":

```
M-x replace-string
C-q C-j
, 
```

Delete indentation

`M-^`

Paste in the search mini-buffer

`C-s M-y`
