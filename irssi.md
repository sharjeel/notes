Irssi
===

Connect to server
`/connect irc.freenode.net 6697`

Set nick
`/nick newnick`

Join a channel
`/join #channel`

Change window 
`M-1 M-2 M-n`

Message somoene
`/msg username message`

Show users in the channel
`/n`

Leave the channel
`/part`

Switch between servers
`C-x`

Window management
`/window stick off`
`/win stick off`
`/win show 2`
`/win m u`
`/win m d`

Help
`/help`
